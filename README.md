# Terminal Text Cleaner

Paste text copied from a terminal and turn messy formatting clean. It rejoins lines that got chopped mid-sentence back into full paragraphs, rebuilds numbered and bulleted lists, and copies the result as rich text or plain text.

**Live tool:** https://chasingnext.com/tools/terminal-text-cleaner/

## What it does

Copying text out of a terminal usually mangles it. Every line gets hard-wrapped, paragraphs break mid-sentence, and lists collapse into loose lines. This tool fixes that:

- Rejoins wrapped lines back into clean paragraphs
- Rebuilds numbered and bulleted lists into real lists
- Copies as rich text (pastes as formatted lists into docs, email, Notion) or as plain text
- Resizable panes and a live preview

## Privacy

Everything runs in your browser. This is a single HTML file with no backend, no analytics, and no tracking. Nothing you paste is captured, stored, or sent anywhere.

## Run it locally

Download this repository and open `index.html` in any browser. No build step, no dependencies, works offline.
